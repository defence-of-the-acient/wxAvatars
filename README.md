# 微信好友头像拼接
获取好友头像并按照定义的图片比例和排序模式拼图
# 用法
IMAGE_WIDTH = 拼图横向宽度最大值（实际输出图片像素宽度为单个头像像素宽度的整数倍且横向宽度不超过此值的最大值/不然会出现黑边）<br/>
IMAGE_RATIO = 画面高宽比例，数值约大画面纵向越长<br/>
user = 保存路径文件夹名
